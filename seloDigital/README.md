<h1 align="center">Desafio de Conhecimento Tecnico VHL<br/>
  API - Selo Digital de Fiscalização - TJSC</h1>
<br/>
Utilizando o Manual de Interoperabilidade, do portal de Selo Digital de Fiscalização - TJSC (http://selo.tjsc.jus.br/html/padrao_tecnico.html), criar uma API em Java com Spring, que apresente os dados de retorno do método getEntesDeclaradosUtilidadePublicaEstadual, com as seguintes funcionalidades:
<br/>
<ol>
  <li>Realizar a operação de login com os dados de homologação descritos no manual;</li>
  <li>Apresentar o retorno do método em um json com o código, a lei e o nome da entidade;</li>
  <li>A API deve permitir a paginação dos dados e realizar a filtragem dos dados com base no id ou no nome da entidade;</li>
  <li>Criar testes unitários para cada cenário mencionado acima.</li>
 </ol>
